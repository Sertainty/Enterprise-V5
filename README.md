## Overview

The **Enterprise Edition** will be the full Sertainty UXP toolset for production deployments. It builds on the [Developers Edition](https://github.com/Sertainty/Developer-V5), adding services, UXL scripting, and utility applications for operating data-centric protection at scale.

> **Downloads will be open — usage is licensed.** On release, anyone will be able to download the SDK; running it will require a separate Sertainty license (evaluation, development, or production). See [Licensing](#licensing).

## What it will include

Everything in the Developers Edition:

- Core UXP libraries
- Supported APIs (C++, C)
- API documentation
- Basic code examples
- Sertainty Technology Guide
- Knowledge Base (offline copy included in the build)
- Additional supported language wrappers

**Plus, exclusive to Enterprise:**

- Services and UXL scripting capabilities
- Utility applications
- Sertainty Services Guide
- Sertainty Scripting Guide

> **Note:** A snapshot of the Knowledge Base will be bundled in each release for offline use. The most current Knowledge Base is always on GitHub — https://sertainty.github.io/knowledge-base/

## Availability

The Enterprise Edition is **not yet available for download.** When it ships, binaries for **macOS, Windows, and Linux** will be published on the Releases page, each with a SHA-256 checksum for verification.

To request early evaluation access or to be notified when v5.1 is released, contact **tech-support@sertainty.com**.

In the meantime, you can preview the **[Enterprise Data Sheet](https://sertainty.github.io/Enterprise-V5/)** (or **[download the PDF](https://sertainty.github.io/Enterprise-V5/Sertainty-Enterprise-v5.1-DataSheet.pdf)**) for capabilities and specifications.
